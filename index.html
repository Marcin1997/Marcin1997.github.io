Who am I?

Front-end developer

Hi, I'm Marcin Grzybowski, I'm a 24 year old self-taught aspiring Front-end developer. I've started my journey with front-end development in early 2021, I've been on this wonderful adventure ever since then.

When I'm not programming I spend lots of my time making jewellery, cycling and generally being surrounder by nature and its wonders!

Ever sine I was little technology has been one of my passions. I love working with CSS and Javascript to make interactive and interesting web pages. I greatly enjoy the wonderful feeling of accomplishment when I finally find a solution for a challenge that I set myself. Drawing App

This is a fully functional drawing app that I made using HTML, CSS3 and Javascript and using Canvas API. In this version of this project you can set a background color, change the color and thickness of the brush, select an eraser and change it's thickness. You can download the drawing if you'd like and lastly you can clear the whole canvas using the clear button

I learned a lot from this project. It helped me work with dimensions a little better while improving my knowledge of Javascript. Creating the brush and its pathing was the biggest challenge. While It also took me some time to seperate the brush from the eraser, the solution was very simple but I think my brain just refused to work in that instance!

Fancy a Quiz?

This is a little quiz I made using HTML5, CSS3 and Javascript. This quiz asks all kinds of questions about me and my interests. From what my favourite dog breed is to my favourite TV show!
